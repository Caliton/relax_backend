## Installation

```bash
$ npm install
```

## Running the app

##### Obs: por enquanto precisa da instalação de um Banco de dados Mysql local... consulte o arquivo .example.env para atribuir os dados de conexão para aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
